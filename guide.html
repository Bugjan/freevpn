🤖 Android

支持 Hiddify / FlClash

適用安裝包架構

  • ARM64：大部分現代 Android 手機
  • ARMv7：較舊 Android 手機
  • Universal：支持多種架構的 Android 設備
  • x86_64：部分基於 x86 的平板或手機

安裝與使用教學

  1. 下載對應架構 APK 並安裝。
  2. 首次啟動授予必要權限（VPN、網路存取）。
  3. 導入配置：掃描 QR Code / 匯入 URL / 手動添加。
  4. 點擊「啟動」按鈕即可連線。
  5. 可切換全局 / 規則 / 自定義模式。

🍎 iOS / iPadOS

支持 Hiddify / Shadowrocket

適用安裝包

  • IPA：適用於 iPhone、iPad、iPod Touch
  • 可使用 AltStore / Sideloadly 或 Xcode 安裝

安裝教學（無越獄）

  1. 使用 AltStore / Sideloadly 安裝 IPA。
  2. 前往「設定 → 一般 → VPN」查看連線狀態。
  3. 啟動 App 並導入配置。
  4. 免費 Apple ID 每 7 天需重新簽名。

安裝教學（越獄設備）

  1. 使用 Xcode 或其他工具直接安裝 IPA。
  2. 完成後打開 App 導入配置。
  • ⚠️ 越獄可能影響保修並增加安全風險。
  • ⚠️ 設備需信任 Apple ID 才能打開 App。

💻 Windows

適用安裝包架構

  • x64 安裝程式（.exe）：支持 Windows 7/8/10/11
  • x64 便攜版（.zip）：無需安裝，直接解壓運行
  • MSIX 安裝包：支持 Windows 10 以上

安裝與使用

  1. 下載並安裝或解壓便攜版。
  2. 首次啟動授權防火牆存取。
  3. 導入配置檔（QR / URL / 手動）。
  4. 點擊「啟動」連線 VPN。

🍏 macOS

適用安裝包架構

  • Intel pkg / dmg：配備 Intel CPU 的 Mac
  • Universal dmg：支持 Intel + Apple Silicon (M1/M2)

安裝與使用

  1. 下載安裝包並安裝。
  2. 開啟 App，導入配置（QR / URL / 手動）。
  3. 允許建立 VPN 連線的權限。
  4. 點擊「啟動」開始使用。

🐧 Linux

適用安裝包架構

  • AppImage x64：支持大多數 Linux 發行版
  • Debian .deb x64：Ubuntu / Debian 系統
  • RPM .rpm x64：Fedora / CentOS / RHEL

安裝與使用

  1. 下載對應包並給予執行權限（例如 chmod +x Hiddify-Linux-x64.AppImage）。
  2. 執行 App 並導入配置。
  3. 點擊「啟動」連線 VPN。

🧭 配置方式與注意事項

  • QR Code：用裝置相機掃描二維碼
  • URL：貼上提供者的配置網址
  • 手動導入：將 config.json 或其他配置檔加入 App
  • 部分系統需手動允許 VPN / 代理權限
  • 若無法連線，請檢查系統時間、配置、以及防火牆 / 安全軟體
  • 中國地區 iOS / MIUI 系統可能對第三方 VPN 有限制
