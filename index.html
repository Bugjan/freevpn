支持的安裝包架構及適用設備

請根據您的設備選擇適合的安裝包架構進行下載：

  • Android：
    • ARM64/Armv8 架構：適用於大部分現代 Android 設備（如許多高端智能手機）
    • ARMv7 架構：適用於較舊的 Android 設備
    • Universal 通用版（支持多架構）：適用於多種不同架構的 Android 設備（例如，舊款與新款設備）
    • x86_64 架構：適用於某些特定的 Android 平台（如基於 x86 的平板或手機）
  • Linux：
    • Debian x64 架構：適用於運行 Debian 或基於 Debian 的 Linux 發行版（如 Ubuntu）的 x64 架構設備
    • 通用 x64 架構（AppImage 格式）：適用於大多數現代 Linux 發行版（如 Ubuntu、Fedora、CentOS）
    • RPM x64 架構：適用於 CentOS、Fedora、RHEL 等 RPM 類型的 Linux 發行版
  • macOS：
    • Intel 架構安裝包（pkg 格式）：適用於配備 Intel 處理器的 Mac 電腦
    • Intel 或 Apple Silicon 架構磁碟映像（dmg 格式）：適用於所有現代 Mac 電腦（包括 Apple Silicon M1/M2 系列處理器的 Mac）
  • iOS：
    • iOS 安裝包（.ipa 格式）：適用於 iPhone、iPad 及 iPod Touch 等蘋果移動設備
  • Windows：
    • 便攜版 x64 架構（zip 格式）：適用於所有支持 x64 架構的 Windows 設備，可直接運行無需安裝
    • 安裝程式 x64 架構（exe 格式）：適用於 x64 架構的 Windows 設備（如 Windows 7/8/10/11）
    • MSIX 安裝包 x64 架構：適用於 Windows 10 或更新版本的設備，支持 MSIX 安裝格式

.ipa 文件安裝教程

.ipa 是蘋果 iOS 應用程序的安裝包格式，用於 iPhone、iPad 和 iPod Touch 等 iOS 設備。以下是如何安裝 `.ipa` 檔案的教程：

安裝方法 1：使用 Cydia Impactor 安裝（無需越獄）

  1. 下載 Cydia Impactor：前往 Cydia Impactor 官方網站 下載並安裝相應版本。
  2. 連接 iOS 設備：使用 USB 線將您的 iPhone、iPad 或 iPod Touch 連接到電腦。
  3. 啟動 Cydia Impactor：運行 Cydia Impactor，確認軟體已識別您的設備。
  4. 拖放 .ipa 文件：將下載的 `.ipa` 檔案直接拖放到 Cydia Impactor 窗口中。
  5. 登錄 Apple ID：輸入您的 Apple ID 和密碼以進行身份驗證。如果您有兩步驟驗證，可能需要生成應用密碼。
  6. 安裝應用：等待 Cydia Impactor 完成安裝過程。安裝完成後，應用將顯示在您的設備主屏幕上。
  7. 信任應用：在您的設備上，前往“設置”->“通用”->“設備管理”並信任您的 Apple ID，然後打開應用。

安裝方法 2：使用 AltStore（無需越獄）

  1. 下載並安裝 AltStore：前往 AltStore 官方網站 下載並安裝 AltStore。
  2. 安裝 AltServer：將 AltServer 安裝到您的電腦（Mac 或 Windows）。安裝後，啟動 AltServer 並在設備上設置 AltStore。
  3. 將設備與電腦連接：將 iPhone 或 iPad 連接到電腦，並確保 AltServer 可以識別您的設備。
  4. 使用 AltStore 安裝 .ipa 文件：將 `.ipa` 文件拖放到 AltStore 中，點擊安裝。
  5. 信任應用：完成安裝後，請到“設置”->“通用”->“設備管理”中信任您的 Apple ID。

安裝方法 3：使用 Xcode 安裝（需越獄）

如果您的設備已經越獄，您可以使用 Xcode 或第三方工具直接將 `.ipa` 文件安裝到您的設備中。

  1. 開啟 Xcode：將 `.ipa` 檔案加載到 Xcode 中，並選擇您的設備作為目標進行安裝。
  2. 安裝應用：Xcode 將自動將應用安裝到設備上。

注意事項

  • 信任應用：在 iOS 9 及以上版本，安裝的應用可能需要您手動信任，請在設置->通用->設備管理中完成此操作。
  • Apple ID 限制：使用免費的 Apple ID 安裝的應用通常只能運行 7 天，過後需要重新安裝。
  • 越獄風險：越獄可能會影響設備的保修，並使其容易遭受安全風險，請謹慎考慮。

    Hiddify 是一款跨平台的代理工具，支援 Android、iOS、Windows、macOS 和 Linux 系統。以下是各平台的基本使用說明：

    📱 Android

    1. 下載對應架構的 .apk 安裝包並安裝（建議使用 ARM64 或 Universal 版本）。
    2. 首次啟動時，授予必要權限（例如 VPN 權限）。
    3. 導入配置：您可以使用 掃描 QR Code、導入配置 URL 或 手動添加。

    📱 iOS

    1. 下載 .ipa 檔案並使用 AltStore、Sideloadly 或其他工具安裝至 iPhone/iPad。
    2. 前往「設定」→「一般」→「VPN」查看連線狀態。
    3. 啟動 Hiddify App，匯入配置並啟動服務。

    注意：未越獄的設備需每 7 天重新簽名（免費 Apple ID）。

    🖥️ Windows

    1. 下載 .exe 安裝包並安裝，或下載 .zip 便攜版直接解壓運行。
    2. 首次啟動可能需要授權防火牆存取。
    3. 點擊「導入配置」→ 選擇配置檔（可使用 QR Code 或 URL）。
    4. 點擊「啟動」以連接代理。

    🖥️ macOS

    1. 下載 .dmg 或 .pkg 安裝包，完成安裝。
    2. 開啟 Hiddify App，導入配置（QR、URL 或手動）。
    3. 如出現權限提示，請允許建立 VPN 連線。
    4. 點擊「啟動」開始使用。
