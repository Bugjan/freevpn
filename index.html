📱 FlClash 使用教學（Android）

  1. 從 GitHub Releases 頁面下載適合你手機架構的 APK 並安裝。
  2. 打開 FlClash，點下方欄「配置」→右下方「+」，匯入你的 Clash 設定檔（檔案、URL 或 QR code）。
  3. 設定好後回到主畫面，點擊右下方按鈕，允許 VPN 權限。
  4. 確認按鈕狀態為「運行中」，即可正常使用。
  5. 如需調整代理模式（全局、規則、自定義），可於主頁右下方切換。
